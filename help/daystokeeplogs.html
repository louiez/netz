Days to keep Logs

This is the number of Days Monitor logs will be stored before being cleared by the daily clean up scripts.
Be carful with this ... during testing with 4800 sites, getting pinged every 10 minutes and saving logs for 30 days... The server started to slow down with displaying the logs. Currently all the pages that display logs only go back 14 days...
so saving logs longer than that will only be usefull if you query the database outside NETz
