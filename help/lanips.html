Buttons

There are 4 buttons [P][T][S][W] for Ping, Telnet,SSH and Web.

[P] The ping button will open the ping window and ping the IP in this field.

[T] Telnet button will attempt to open a Telnet session with the default telnet client on your system. you will need to alter the registry to use the correct application for telnet the registry key is HKEY_CLASSES_ROOT\telnet\shell\open\command if you are not sure how to do this get with your PC support

[S] the SSH button will open the application used with SSH URL handler from a browser normally windows dosen't have a URL handler for SSH and you will need to add one just like the Telnet URL handler located at HKEY_CLASSES_ROOT\

sample reg import file that uses putty.. just change if you use a different app for ssh client

[W] Web button will attempt to web into the site ie: ( http://FIELD-IP ) from the web browser
