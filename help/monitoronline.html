Monitor Online check IPs

Monitor IPs are used to check the NETz monitor to be sure it is online.

If it can't ping ANY of these addresses it will assume it is offline and not send any alerts for sites offline.
But It will try to send email to the NETz admins....
assuming that the NETz server is offline... these emails might never make it to the Admins....but it will try anyway.. :-)
