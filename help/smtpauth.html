SMTP Auth

Many providers now block outbound port 25 (email) traffic.
in order to send mail you must relay off your providers email server.
then this relay started to become an issue so now most will require you
to not only relay off their server but logon first with a valid account using SMTPAUTH.
some providers even go one step further, and require a valid "authorized" from email address.

Now with that rant done.
If you want NETz to send alert emails and you have a provider with SMTPAUTH requirements.
Netz provides a place to enter a valid User, password and from address.
